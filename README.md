# 🦎 LizardKTS (Keep This Shit)

Quality-of-life addon for Synastria that reveals item upgrade chains to prevent accidental vendoring or deletion before attuning.

Built specifically for the Synastria progression system.

---

## ✨ Features

- 🔗 Shows full upgrade chains
- 🚨 Prevents accidental item loss
- 🧠 Synastria-aware logic
- 🪶 Lightweight and unobtrusive
- ⚡ Instant tooltip integration

---

## 🎯 Why This Exists

On Synastria, many items upgrade into future attunements.

Without visibility, it's easy to:

- ❌ Vendor important items  
- ❌ Delete future upgrades  
- ❌ Break attunement chains  

**LizardKTS solves this by showing upgrade paths directly in-game.**

---

## 🕹️ How It Works

Simply install the addon.

When hovering items, the addon will:

- Detect upgrade chains  
- Display relevant progression info  
- Help you decide what to keep  

No configuration required.

---

## 🧩 Supported Client

- ✅ WotLK 3.3.5
- ✅ Synastria private server

---

## 📦 Installation

1. Download latest release
2. Extract into: Interface/AddOns/
3. Ensure folder structure is: Interface/AddOns/LizardKTS/

4. Restart the game
